# 서울시 지하철 실시간 위치 조회 앱

서울특별시 교통정보 시스템(TOPIS) API를 사용하여 지하철 실시간 열차 위치를 조회하는 React 웹 애플리케이션입니다.

🌐 **라이브 데모**: [https://kccho88.github.io/Subway-location](https://kccho88.github.io/Subway-location)

## 기능

- 🚇 **호선별 검색**: 서울시 지하철 호선 선택 (1호선~9호선, 경의중앙선, 공항철도 등)
- 🏢 **역별 검색**: 역 이름으로 검색하여 해당 역을 지나는 모든 열차 정보 조회
- 📍 실시간 열차 위치 정보 표시
- 🔄 수동/자동 새로고침 (30초 간격)
- 📊 페이징 지원 (더 보기 기능)
- 📱 반응형 디자인
- ⚠️ 오류 처리 및 사용자 친화적 메시지

## 설치 및 실행

### 필수 요구사항

- Node.js (v14 이상 권장)


배포 후 [https://kccho88.github.io/Subway-location](https://kccho88.github.io/Subway-location)에서 확인할 수 있습니다.

## 사용 방법

### 호선별 검색
1. 상단의 "🚇 호선별 검색" 탭을 선택합니다.
2. 드롭다운에서 지하철 호선을 선택합니다.
3. 선택한 호선의 실시간 열차 위치 정보가 자동으로 표시됩니다.

### 역별 검색
1. 상단의 "🏢 역별 검색" 탭을 선택합니다.
2. 역 이름을 입력합니다 (예: 서울역, 강남, 종로3가).
3. 검색 버튼을 클릭하거나 Enter 키를 누릅니다.
4. 해당 역을 지나는 모든 호선의 열차 정보가 표시됩니다.

### 공통 기능
- "새로고침" 버튼을 클릭하여 수동으로 데이터를 갱신할 수 있습니다.
- "자동 새로고침 시작" 버튼을 클릭하면 30초마다 자동으로 데이터가 갱신됩니다.
- 호선별 검색에서 데이터가 많을 경우 "더 보기" 버튼을 클릭하여 추가 데이터를 로드할 수 있습니다.

## API 정보

- **API 제공**: 서울특별시 교통정보과(TOPIS)
- **API 키**: 774a4e43776b6363363248655a6b42
- **API URL**: `http://swopenAPI.seoul.go.kr/api/subway/{KEY}/json/realtimePosition/{START_INDEX}/{END_INDEX}/{subwayNm}`

## 데이터 필드 설명

- **열차 번호**: 열차의 고유 번호
- **현재 역**: 열차가 현재 위치한 역 이름
- **방향**: 상행/내선(0) 또는 하행/외선(1)
- **상태**: 진입(0), 도착(1), 출발(2), 전역 출발(3)
- **종착역**: 열차의 최종 목적지 역
- **급행 여부**: 일반, 급행(1), 특급(7)
- **막차**: 막차 여부 (예/아니오)
- **수신 시간**: 데이터 수신 시간

## 참고사항

- API는 CORS 정책에 따라 브라우저에서 직접 호출 시 차단될 수 있습니다. 이 경우 프록시 서버를 사용하거나 백엔드 서버를 통해 API를 호출해야 할 수 있습니다.
- API 응답 시간에 따라 데이터 로딩이 지연될 수 있습니다.
- 일부 호선의 경우 실시간 데이터가 제공되지 않을 수 있습니다.
